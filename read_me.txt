Установка Python

1. Скачай и установи Python: https://www.python.org/downloads/
Установка простая: далее-далее-готово. Но! В первом окне установщика поставь галку "Добавить в PATH". Перезагрузи комп.

2. Открой командную строку (WIN + R, cmd, OK), напиши там python pip install Openpyxl и нажми enter

Работа со скриптом

1. Проверь, чтобы в пути нахождения скрипта не было пробелов. С ними что-то не работает

2. Открой командную строку (WIN + R, cmd, OK), напиши там python ПУТЬ_К_СКРИПТУ/main.py и нажми enter
